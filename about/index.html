关于

语言无止境，框架海无涯。

虽然早在2013年就写下了第一篇博文，但其后很多年都没有坚持下来，直到2017年域名与空间全都到期（国外的廉价空间并不怎么好用）。加之以前的域名也不是太喜欢，而且 WordPress 也用腻了，所以决定在 GitHub 重新开始。

至于以前的文章，相当一部分现已经过时，就让它们随风去吧（反正也没什么人看😅）；只保留一些值得收藏的文章，逐一迁移过来。

感谢 GitHub 提供免费好用的 GitHub Pages；同时也感谢 Hexo 和 NexT 的开发者们。

PS: 本博最先用的网站生成器是 Hugo + Even，为此还写了一篇搭建教程，后来发现 Hexo + NexT 更符合我的口味，遂换之。

博主

Zil Herig，昵称 Zilor，出生于80年代末期。NET 开发工程师,.NET 架构师,.NET 死忠粉。

8岁开始学习编程的我，现已经成长为油腻大叔，每每回顾身后，发现这一路尽是曲折、坎坷、艰辛和迷茫……竟然一次狗屎运都没有踩过。

哪怕掌握了 VB、C++、Delphi、PHP、C#、Pb、Python 等在内的十八般语言，并且样样精通（吹个牛B行不行！），也依然深深的感受到了这个世界的不友好——语言无止境，框架海无涯。

作为芸芸众生中的平凡人（对，我早已承认了自己的平凡😕），我们能做的，就是跟紧脚步，不敢懈怠。

因为——

犹豫，就会败北

博客

我喜欢清净，又不喜欢太清静。

这就像我喜欢把文章记录在博客中，但又不喜欢在 XX园、cnbxxx 这些文人极客聚集的地方写。

我喜欢找一隅偏安，记录着自己的学习历程、收藏着有价值的文章，或许还能为有缘的路过之人提供些许价值，换得只言片语的感谢……

这就是本博的宗旨。
